# SwiftBlogger  <a href="url"><img src="public/logo.PNG" align="left" height="48" width="48" ></a>
This project is deployed to https://swiftblogger.onrender.com

A simple blogging site made using Node.js, Express.js, and EJS. Posts are not persistent between sessions as no database are used in this version of the application.
![Screenshot 2024-03-28 200037](https://github.com/marcelmunaba/SwiftBlogger/assets/70313979/39b41400-f751-4651-9abd-a400cad339bc)

# Features
**1. Post Creation**: Users are able to create new posts.
![Screenshot 2024-03-28 200403](https://github.com/marcelmunaba/SwiftBlogger/assets/70313979/95405a4a-a68d-419a-b5cc-7159af5e84bd)

**2. Post Viewing**: The home page allows the user to view all their posts.
![Screenshot 2024-03-28 200437](https://github.com/marcelmunaba/SwiftBlogger/assets/70313979/e5c9f1ae-2142-4d72-b99d-2e175e8e67f9)

**3. Post Update/Delete**: Users can edit and delete posts as needed.
![Screenshot 2024-03-28 200550](https://github.com/marcelmunaba/SwiftBlogger/assets/70313979/0aaa888c-db35-41e0-9810-fa8f43f9de05)
