# SwiftBlogger  <a href="url"><img src="public/logo.PNG" align="left" height="48" width="48" ></a>
This project is deployed to https://swiftblogger.onrender.com

A simple blogging site made using Node.js, Express.js, and EJS. Posts are not persistent between sessions as no database are used in this version of the application.
![home](https://github.com/marcelmunaba/SwiftBlogger/assets/70313979/c8310cf9-5395-4a01-99b8-fa6d7fc9740a)


# Features
**1. User Authentication**: Users are able to create accounts and log in.
![Landing_page](https://github.com/marcelmunaba/SwiftBlogger/assets/70313979/d2ab069b-73ab-499d-834d-46d81d10cbcb)
![sign_up](https://github.com/marcelmunaba/SwiftBlogger/assets/70313979/9606c132-afcf-438f-84c0-3b45eafdbe0d)
![login](https://github.com/marcelmunaba/SwiftBlogger/assets/70313979/990a525c-e829-4a13-8fea-53b843a17460)

**2. Post Creation**: Users are able to create new posts.
![create_post](https://github.com/marcelmunaba/SwiftBlogger/assets/70313979/b00f2095-6016-4289-95cc-1e9b049563ba)

**3. Post Viewing**: The home page allows the user to view all their posts.
![blogposts](https://github.com/marcelmunaba/SwiftBlogger/assets/70313979/d9873cb8-ccd5-4376-8987-945318242afe)

**4. Post Update/Delete**: Users can edit and delete posts as needed.
![edit](https://github.com/marcelmunaba/SwiftBlogger/assets/70313979/41276726-ccee-4688-991a-951a59d09c2c)
